# Student Attendance System

A web-based Student Attendance System developed using Java, Servlets, JSP, and MySQL.  
This project helps manage student records and attendance in an efficient and organized way.

---

## Project Description

The Student Attendance System is designed to automate the process of managing student attendance.  
It reduces manual work and provides an easy-to-use interface for maintaining attendance records.

This project is developed as part of an academic requirement and follows basic web application architecture.

---

## Features

- User login system
- Student record management
- Attendance marking
- Attendance viewing
- Simple and user-friendly interface
- Frontend and backend integration

---

## Technologies Used

### Frontend

### Backend

- MySQL

## How to Run the Project

1. Import the project into Eclipse
2. Configure Apache Tomcat Server
3. Create the MySQL database
4. Update database configuration locally
5. Run the project on the server
6. Access the application through a web browser
